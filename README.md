### Диаграмма прецедентов ###

  - `usecase (Делать что-то) as UC`
  - `usecase "Делать что-то" as UC`
- [x] определение узлов `actor`:
  - `actor :Какой-то актор: as A`
  - `actor "Какой-то актор" as A`
- [x] ассоциация, включение, расширения, наследование:
  - `UC <-- A`
  - `UC ..> UC1 : <<include>>`
  - `UC <.. UC1 : <<extend>>`
  - `A1 --|> A`
- [x] группировки `package`/`rectangle`;
